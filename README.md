# Hackathon Video Documentation

## Overview
The **Hackathon Video** project aims to deliver a comprehensive solution for video handling, sharing, and visualization during hackathon events.

### Objectives
- Facilitate seamless video uploads.
- Provide video streaming capabilities.
- Ensure easy access and sharing of videos among participants and stakeholders.

## Repositories
The Hackathon Video project is composed of several repositories, each responsible for different functionalities:

1. **hackathon-video-backend**  
   - Responsible for handling server-side logic, including video processing and database interactions.
   - [Link to Repository](https://github.com/postech-group-fiap-188/hackathon-video-backend)

2. **hackathon-video-frontend**  
   - Manages the user interface, providing an intuitive platform for users to upload and manage videos.
   - [Link to Repository](https://github.com/postech-group-fiap-188/hackathon-video-frontend)

3. **hackathon-video-database**  
   - Contains database schema and migration files crucial for the project.
   - [Link to Repository](https://github.com/postech-group-fiap-188/hackathon-video-database)

## Getting Started
To get started with the Hackathon Video project:
1. Clone the repositories to your local machine.
2. Set up the required environment variables.
3. Follow the setup instructions in each repository for installation and running the applications.
